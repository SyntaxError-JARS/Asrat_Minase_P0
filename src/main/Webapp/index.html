Welcome to Austin Bank Application

This is a banking application project that enable the users to create a new customer, new account, deposit and withdraw fuds from an account. It is working as a backend only. To use this application Postman was used.

As a new customer,

  • I should be able to register on the system, so that I can use the application.
  • I should be able to login once I have registered so that I can access the information in the database.
  • I should be able to create a new Customer
  • I should be able to create an account
  • I should be able to deposit funds into an account
  • I should be able to withdraw funds from an account
  • I should be able to view the balance of my account
  • I should be able to log out from the system

All the above tasks are achievable by sending HTTP Requests to the end points provided in the postman executable

List of tools used for making this banking application
  • Java 8.
  • Apache Maven
  • PostgresSQL
  • Git SCM (hosted on GitHub)
  • HTML/CSS
  • Apache Tomcat 9.0.62 Servlets
  • Postman
